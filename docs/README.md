# Screenshots Directory

This directory contains screenshots and images for the documentation.

## Files:
- `chat-ui-screenshot.png` - Main interface screenshot showing the chat application

## Instructions to add your screenshot:

1. Save your screenshot as `chat-ui-screenshot.png` in this directory
2. The image should show the complete interface with:
   - Chat messages between user and AI
   - Sidebar with connection status
   - Input field at the bottom
   - Clean, modern UI design

Your screenshot will automatically appear in the README.md file once added here.
